## Description
An ecommerce store prototype built with MERN stack, that enables customers to purchase products and utilizes third party API's.


## User Story
```
  AS A manager at an internet retail company
  I WANT a full stack e-commerce website that uses the latest technologies
  SO THAT my company can compete with other e-commerce companies
```


## Acceptance Criteria 
```
  GIVEN e-commerce website
  WHEN I load the e-commerce website
  THEN I am presented with a homepage with the options Search Products, View Wishlist,  and Login/Signup 
  WHEN I click on the Search Products menu option
  THEN I am presented with an input field to search for products  
  WHEN I click on the Login/Signup menu option
  THEN a modal appears on the screen with a toggle between the option to log in or sign up
  WHEN the button is clicked to Signup
  THEN I am presented with inputs for a username, an email address, and a password, and a signup button
  WHEN the button is clicked to Login
  THEN I am presented with two inputs for an email address and a password and login button
  WHEN I enter a valid email address and create a password and click on the signup button
  THEN my user account is created and I am logged in to the site
  WHEN I enter my accountâ€™s email address and password and click on the login button
  THEN I the modal closes and I am logged in to the site
  WHEN I am logged in to the site
  THEN the menu options change to Search for Products and an option to see my saved wishlist
  WHEN I am logged in and enter a search term in the input field and click the submit button
  THEN I am presented with search results and two buttons to add to cart and to add to wishlist
```


## Installation
- Step 1: Clone this repository. you can do this by running ``` git clone git@github.com:adinair007/Mojave.git``` in your terminal
- Step 2: Open your code editor by running the command ``` code . ```
- Step 3: Install dependencies by running the ``` npm install ``` command in the command line


## Setup
- Step 1: Run the ``` npm run seed ``` command in the command line
- Step 2: Run the ``` npm run develop ``` command in the command line
- step 6: Start the server by running ``` npm start ``` in the terminal. Make sure you are in the client folder for this -- This will pull up the website on Localhost 3000
